# Virtual IT Support Home Lab

This repository documents a personal virtualized IT support environment built using Oracle VirtualBox.

- Operating System installation and configuration
- Basic networking setup
- Troubleshooting common technical issues
- Technical documentation and process workflows

---

## Lab Environment

**Hypervisor:** VirtualBox 7.1.10r169112  
**Guest Operating Systems:**
- Windows 10 Enterprise
- windows server 2022
- Ubuntu Linux
- Kali Linux

---

## Lab Objectives

This home lab helps to:
- Understand OS deployment  
- Configure basic network connectivity  
- Document system setups step-by-step  
- Simulate common support tasks such as login or connection issues  
- Track resolutions in an organized format
